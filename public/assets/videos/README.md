# Video Assets

Drop your video files here. The site will auto-use them when named correctly.

## Required Files

| Filename | What it's for |
|---|---|
| `story.mp4` | "Our Story" popup on homepage hero |
| `hero-reel.mp4` | Homepage hero background loop |
| `makers-mark-handcrafted.mp4` | Maker's Mark handcrafted portfolio video |
| `makers-mark-jcole.mp4` | Brooklyn Mirage event video |
| `redbull-daysnatchers.mp4` | Red Bull × Daysnatchers |
| `jase-candy-land.mp4` | JASE animation video |

## Format
- MP4, H.264 codec (best browser compatibility)
- Hero reel: keep under 20MB for load speed
- Story video: any size, streams fine
